SUBIR A GITHUB
1. Extrae este ZIP.
2. En el repositorio Change-Makers, pulsa Add file > Upload files.
3. Arrastra TODO el contenido de esta carpeta, no el ZIP.
4. Confirma que index.html quede en la raíz del repositorio.
5. Pulsa Commit changes.

PUBLICAR EN VERCEL
1. En Vercel: Add New > Project.
2. Importa el repositorio Change-Makers.
3. Framework Preset: Other.
4. Build Command: déjalo vacío.
5. Output Directory: déjalo vacío.
6. Pulsa Deploy.
